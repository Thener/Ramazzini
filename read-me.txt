1. Pré-requisitos:

Java 1.7 (jdk-7u71-windows-x64.exe)
PostgreSql 9.3 (postgresql-9.3.5-3-windows-x64.exe)
postgresql-9.3-1102-jdbc41.jar
Elipse Luna
Jboss (jboss-as-7.1.1.Final)

2. Instalação:

- Faça o clone do projeto: https://github.com/Thener/Ramazzini.git
- Abra o eclipse e use a opção: File > Import > Existing Maven Projects
- No eclipse, no menu; [window > preferences > installed JREs] e verifique se a jre7 está configurada (Caso não esteja, crie apontando para "C:\Program Files\Java\jre7")
- Ainda em "Window > Preferences", pesquise por "Execution Environments", clique em JavaSE-1.7 e marque a jre7 em "compatible JREs"
- Na aba "Servers", adicione o "JBoss AS 7.1" e confirme a runtime para JBoss 7.1 Runtime
- Ainda em "Window > Preferences", pesquise por "Server > Runtime Environments" e clique no botão para editar "Jboss7.1 Runtime"; Em seguida, mude a "Runtime JRE" para "JavaSE-1.7"
- Ajuste o module.xml do postgre no jboss (jboss-as-7.1.1.Final\modules\org\postgresql\main) copiando o arquivo "postgresql-9.3-1102-jdbc41.jar" e acertando o xml.
- Ajuste o standalone.xml em (jboss-as-7.1.1.Final\standalone\configuration), acrescentado as configurações do driver do postgresql. 
- Abra o pgAdmin e crie a base do sistema (ramazzini)
- rode o projeto para criação das tabelas (caso o banco não seja criado, verifique o arquivo persistence.xml, a opção: <property name="hibernate.hbm2ddl.auto" value="validate" />
